## 岗位职责

1. 负责 **PC 端 Web 专业系统**（学术用户核心终端）的前端开发
2. 基于 **React + TypeScript** 搭建前端工程体系，包括状态管理、路由、组件库
3. 与 GIS 工程师协作集成地图组件（Leaflet.js/OpenLayers React 封装）
4. 实现 **多维诗词检索** 的前端交互（6维度组合筛选 + 地图/列表/对比三形态展示）
5. 实现 **AI 创作辅助工具** 的前端交互（对仗推荐输入框、格律标注、意境选配等）
6. 实现 **数据导出** 前端功能（Excel/CSV/shp/PNG 等格式的触发与下载）
7. 负责 **学术图表** 的前端渲染（基于 ECharts/D3.js 的统计图表）
8. 确保 PC 端兼容 Chrome/Edge/Firefox，最低系统 Windows 10

---

## 技术要求

### 硬性要求
- ✅ **3年以上**前端开发经验
- ✅ 精通 **React**（Hooks、Context、性能优化），有大型项目经验
- ✅ 精通 **TypeScript**
- ✅ 熟悉状态管理方案（Redux / Zustand / Jotai）
- ✅ 有与 GIS 地图组件（Leaflet/OpenLayers）集成经验
- ✅ 熟悉 RESTful API 对接、WebSocket 通信

### 加分项
- ⭐ 有 **Leaflet.js/OpenLayers React 封装** 实际项目经验
- ⭐ 有数据可视化（ECharts/D3.js）项目经验
- ⭐ 有 GIS 时空数据展示经验
- ⭐ 了解模块联邦（Module Federation）等微前端方案

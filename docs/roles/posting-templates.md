## 岗位1：后端架构师 / 技术负责人

**标题：Python FastAPI 后端架构师（诗词文化+GIS 方向）**

**公司/项目：** 中国古诗词文化互动平台（PoetrySpace）— 新项目，产品已定，团队组建中

**薪资：** 30K-45K /月（优秀可谈）

**岗位职责：**
1. 负责平台后端架构设计，主导微服务拆分（7类服务 + API Gateway + BFF 层）
2. 主导 PostgreSQL + PostGIS 时空数据库设计（地名/诗人轨迹/诗词标注三大库）
3. 设计 Elasticsearch/OpenSearch 时空复合索引，支撑六维组合检索 ≤500ms
4. 制定 FastAPI 开发规范（RESTful API、JWT 鉴权、统一异常处理）
5. 与 AI/算法团队协作，将 NLP 模型封装为高性能推理服务

**硬性要求：**
- 5年+ 后端开发，2年+ 架构设计经验
- 精通 Python + FastAPI（或类似异步框架）
- 精通 PostgreSQL，有 PostGIS 扩展实际使用经验
- 有微服务 + Docker/K8s 项目经验

**加分项：**
- 有 Elasticsearch 时空索引设计经验
- 有 GIS 相关项目经验
- 有 AI 模型服务化部署经验
- 对数字人文/唐宋文学有兴趣

**技术关键词：** FastAPI · PostgreSQL · PostGIS · Elasticsearch · Docker · Redis · GIS

**面试流程：** 简历筛选 → 技术面（含15分钟架构设计题）→ 终面

## 岗位2：GIS 开发工程师

**标题：GIS 可视化开发工程师（诗词轨迹 + 时空热力分析）**

**公司/项目：** 中国古诗词文化互动平台（PoetrySpace）— 核心差异化功能，产品已定

**薪资：** 25K-40K /月（优秀可谈）

**岗位职责：**
1. 实现诗人行迹动态可视化（三级时间轴 + 飞线动画 + 陆路/水路区分）
2. 实现多诗人同屏对比（2-10位轨迹叠加 + 颜色区分 + 一键显隐）
3. 实现诗词热力时序动态渲染（时间轴拖动 → 热力图渐变更新）
4. 实现交游网络可视化、数据导出（GIS shp、高清PNG ≥300DPI）

**硬性要求：**
- 3年+ 前端/GIS 开发经验
- 精通 Leaflet.js 或 OpenLayers
- 精通 JavaScript/TypeScript + React
- 有 Canvas/SVG/WebGL 绘图经验
- 理解 WGS84、GeoJSON、地图投影等 GIS 基础概念

**加分项：**
- 有轨迹动画/动态渲染项目经验
- 有 D3.js/ECharts 可视化库经验
- 有 PostGIS 空间查询编写经验
- 对唐宋文学有研究者优先（面试时聊聊诗词也是加分）

**技术关键词：** Leaflet.js · OpenLayers · React · TypeScript · Canvas · D3.js · GeoJSON · GIS

**面试流程：** 简历筛选 → 作品展示（看真实项目）→ 技术面

## 岗位3：AI 算法工程师（NLP方向）

**标题：NLP/AI 算法工程师（古典诗词方向 — 对仗/格律/意境生成）**

**公司/项目：** 中国古诗词文化互动平台（PoetrySpace）— AI 创作辅助全链路

**薪资：** 30K-50K /月（优秀可谈）

**岗位职责：**
1. 对仗推荐引擎：基于 265 万组对仗词库，实现 AI 实时推荐（≤500ms）
2. 格律校验引擎：全体裁覆盖（绝句/律诗/词牌），双韵库（平水韵/中华新韵）
3. 意境匹配与文本生成：意境框架生成、诗词解析、仿写改写、多意境融合
4. 将模型封装为 FastAPI 高性能推理服务

**硬性要求：**
- 3年+ NLP/AI 算法经验
- 精通 Python + PyTorch/Transformers
- 有文本生成/序列标注/文本分类实际项目经验
- 了解中文古典诗词格律基础（平仄、押韵、对仗）
- 有模型推理性能优化经验（ONNX/量化/蒸馏等）

**加分项：**
- 有古典诗词 NLP 项目经验（诗词生成、格律检测、对仗匹配）
- 有知识图谱/语义网络构建经验
- 有 LLM 微调/Prompt Engineering 经验

**特别说明：** 面试会有诗词格律基础考察 + 技术方案设计题。我们不要求你已经是诗词专家，但要对这个方向有热情。

**技术关键词：** NLP · PyTorch · Transformers · ONNX · FastAPI · RAG · LLM

**面试流程：** 简历筛选 → 技术面（含诗词基础+方案设计）→ 终面
